## 🛠️ Getting Started

### Clone the Repository
```bash
git clone https://github.com/Shreyeah03/portfolio.git
cd portfolio
```

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio.
